# flutter_my_yoga

这是一个练习项目，只做了页面排版，不包含业务逻辑，在 IPhone 8 下开发，其他型号页面显示可能有问题。

UI 设计来自 [http://freebielobster.com](http://freebielobster.com/index.php/home/post/Yoga-App-Ui-Freebie)，仅供学习使用。
